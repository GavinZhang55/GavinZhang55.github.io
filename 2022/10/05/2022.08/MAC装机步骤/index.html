Homebrew简介

  • homebrew 是MacOS 的软件包管理器。对各种应用软件进行管理，如安装，更新，查看，搜索，卸载，等等。

  • 官网地址为

  • Homebrew 3.0.0版开始，开始支持Apple M1芯片。

为什么安装Homebrew

  • 在终端命令行中，输入安装命令

    • 官网安装命令（必须有VPN， 不推荐）

    • 国内镜像安装（无VPN， 推荐）

      • 安装命令

      • 执行后，会有操作提示

      1. 选择下载镜像（推荐，2、清华大学下载源）

      2. 确认删除旧版本

      3. 输入开机密码（用于mac确认第三方应用安装）

      4. 安装git

        • 如果之前没有安装过git， 会终止Homebrew安装，弹出git 安装提示，选择安装

        • git 全称是分布式版本控制系统， 项目版本管理工具。

      5. 再次执行Homebrew安装命令

      6. 验证是否安装成功

        • 在终端输入”brew -v“（homebrew, 在终端中输入brew即可）, 可查看Homebrew版本。

Node 安装

Node 简介

  • Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时环境

    • chrome V8引擎，为了将JS代码编译成CPU可识别的汇编代码

为什么要安装Node

  • 因为npm（或者yarn）的应用基于node.js，npm（或者yarn）是Node.js的包管理工具（package manager）

  • npm

    • 前端开发，会用到别人写的JavaScript代码，如果我们要使用别人写的某个包，每次都根据名称搜索一下官方网站，下载代码，解压，再使用，非常繁琐。于是使用一个集中管理的工具npm，把开发的模块打包后放到npm官网上，要使用，直接通过npm安装就可以直接用。更重要的是，如果我们要使用模块A，而模块A又依赖于模块B，模块B又依赖于模块X和模块Y，npm可以根据依赖关系，把所有依赖的包都下载下来并管理起来

  • yarn

    • Yarn 是为了弥补 npm 的一些缺陷而出现的，比如提高安装速度，安装版本统一等等

Node安装步骤

  • 使用Homebrew在终端内安装Node，命令如下

  • 安装node之后，可以切换npm镜像，提速npm
